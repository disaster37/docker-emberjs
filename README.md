# docker-emberjs
## Overview
The container include some usefull tools to code with EmberJS framwork.

It's include :
* NodeJS 0.12.x
* NPM
* Bower
* Ember-cli
* Webstorm IDEA

## Quick start

```bash
docker-compose up -d
```

It will open thow shell that you can use to send some ember-cli command and to serve your application.
It will also open Webstorm.
